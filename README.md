# PiGreco
A software in C++ for calculate the PiGreco

# HowTo Install?
  MAC OS<br>
    1. Install "http://brew.sh" by following the instructions on the website<br>
    2. Type in terminal the following lines:<br>
       <pre>
       $ brew install make
       $ brew install gcc
       $ brew install cc
       </pre>
    3. Then type "./install_macos"<br><br>
    
    Now to run "./pigreco"
